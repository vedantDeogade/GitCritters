## 🏗️ Core Tech Stack

### 1. **Language & Runtime**

| Component | Tool | Version | Purpose |
|---|---|---|---|
| **Runtime** | Node.js | 14+ (16+ recommended) | Executes extension code |
| **Language** | TypeScript | 4.7+ | Type-safe extension development |
| **Transpiler** | tsc (TypeScript Compiler) | bundled | Converts TS → JS |

**Install TypeScript:**
```bash
npm install --save-dev typescript@latest
```

**tsconfig.json:**
```json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "commonjs",
    "lib": ["ES2020"],
    "outDir": "./out",
    "rootDir": "./src",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true
  },
  "include": ["src"],
  "exclude": ["node_modules", "out"]
}
```

### 2. **VS Code APIs**

| Package | Version | What It Does |
|---|---|---|
| `@types/vscode` | 1.70+ | Type definitions for VS Code API |
| `vscode` | built-in | Runtime module for extension commands, panels, etc. |

**Install:**
```bash
npm install --save-dev @types/vscode@latest
```

**Key APIs Used:**
- `vscode.extensions.getExtension()` — access Git API
- `vscode.window.createWebviewPanel()` — create animation panels
- `vscode.commands.registerCommand()` — register git-related commands
- `vscode.workspace.getConfiguration()` — read user settings
- `vscode.window.showInformationMessage()` — show notifications

### 3. **Module Bundler (Webpack)**

| Package | Purpose |
|---|---|
| `webpack` | Bundles TypeScript modules into single JS file |
| `webpack-cli` | CLI tool for webpack |
| `ts-loader` | Integrates TypeScript with webpack |

**Install:**
```bash
npm install --save-dev webpack webpack-cli ts-loader
```

**webpack.config.js (Auto-generated by yo code):**
```javascript
const path = require('path');

module.exports = {
  target: 'node',
  mode: 'production',
  entry: './src/extension.ts',
  output: {
    path: path.resolve(__dirname, 'out'),
    filename: 'extension.js',
    libraryTarget: 'commonjs2'
  },
  externals: {
    vscode: 'commonjs vscode'
  },
  resolve: {
    extensions: ['.ts', '.js']
  },
  module: {
    rules: [
      {
        test: /\.ts$/,
        use: 'ts-loader',
        exclude: /node_modules/
      }
    ]
  }
};
```

## 🎨 Client-Side (WebView) Tech Stack

For animations and mini-games running inside WebviewPanels:

### Canvas & Graphics

| Technology | Purpose | Native? |
|---|---|---|
| **HTML5 Canvas API** | Draw pixel art, animations | ✅ Built-in |
| **Canvas 2D Context** | Pixel-by-pixel drawing | ✅ Built-in |
| **requestAnimationFrame()** | Smooth animation loop | ✅ Built-in |
| **image-rendering: pixelated** | CSS for authentic 8-bit look | ✅ Built-in |

**Example Canvas Setup:**
```html
<canvas id="animationCanvas" width="340" height="160" 
        style="image-rendering: pixelated;">
</canvas>

<script>
  const canvas = document.getElementById('animationCanvas');
  const ctx = canvas.getContext('2d');
  
  // Draw pixel-art cat
  ctx.fillStyle = '#ff9933';
  ctx.fillRect(x, y, 16, 16); // 16x16 pixel cat
  
  requestAnimationFrame(() => draw());
</script>
```

### Audio (8-Bit Sound)

| API | Purpose | Alternative |
|---|---|---|
| **Web Audio API** | Generate square-wave synth sounds | Howler.js (extra lib) |
| **OscillatorNode** | Create 8-bit tones | N/A |
| **GainNode** | Control volume / fades | N/A |

**Example 8-Bit Audio:**
```javascript
function playPush() {
  const ctx = new AudioContext();
  const osc = ctx.createOscillator();
  const gain = ctx.createGain();
  
  osc.type = 'square'; // 8-bit waveform
  osc.frequency.value = 523; // C5 note
  
  osc.connect(gain);
  gain.connect(ctx.destination);
  
  osc.start(ctx.currentTime);
  osc.stop(ctx.currentTime + 0.2);
}
```

### Content Security Policy (CSP)

| Requirement | Why |
|---|---|
| **Nonce-based scripts** | VS Code enforces strict CSP; inline scripts need cryptographic nonce |
| **webview.cspSource** | Reference for resource origins (images, stylesheets) |
| **Meta tag** | CSP rules must be in a `<meta>` tag, not headers |

**WebView CSP Template:**
```html
<meta http-equiv="Content-Security-Policy" 
      content="default-src 'none'; 
               img-src ${webview.cspSource}; 
               script-src 'nonce-${nonce}';">

<script nonce="${nonce}">
  // Animation code here
</script>
```

## 🔄 Build & Development Workflow

### npm Scripts (in package.json)

```json
{
  "scripts": {
    "vscode:prepublish": "webpack --mode production",

}
```

**Common Commands:**

| Command | What It Does |
